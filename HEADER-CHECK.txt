WHAT I'M LOOKING FOR

zoom    should be identical on both pages. If it isn't, that's the whole
        problem — press Ctrl+0 on each page to reset and re-check.
win     the window width. Must match on both.
doc     the document width. If this is BIGGER than "win" on one page, that page
        can scroll sideways, which makes everything on it look shifted left.
root    the root font size. Must be identical — everything sized in rem keys
        off this.
hdr     left/top/width of the header block. These three numbers are the answer:
        if they match on both pages, the header is not moving and something
        else is (scroll position, zoom). If they differ, I can work out
        precisely which rule is responsible from the numbers alone.
logo    rendered crest height in px. Should be the same on both.
title   computed size of the EquiStories wordmark. Should be the same on both.
css     which stylesheets actually loaded, in order. If one page is missing a
        file, or loading an extra one, it shows up here.
